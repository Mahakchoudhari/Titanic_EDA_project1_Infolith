# Titanic EDA Project

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python and Jupyter Notebook.

## Goal

Understand which factors influenced passenger survival on the Titanic.

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure

titanic-eda/

├── data/

│ └── train.csv

├── notebook/

│ └── titanic_eda.ipynb

├── requirements.txt

├── README.md

└── .gitignore

## Key Analysis

The notebook explores:

- Missing data
- Passenger demographics
- Survival distribution
- Survival by gender
- Survival by passenger class
- Age distribution
- Fare analysis

## Findings

Women and higher-class passengers had significantly higher survival rates. Third-class passengers experienced the highest mortality. Children showed better survival chances than older adults, supporting the “women and children first” evacuation behavior.

## How to Run

1. Clone the repository

git clone <your-repository-link>

2. Install dependencies

pip install -r requirements.txt

3. Open Jupyter Notebook

jupyter notebook

4. Run all cells from top to bottom.
